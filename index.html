Portfolio 01

LG SIGNATURE (Global page) 포트폴리오 보러가기

제작 의도

포트폴리오 제작을 위해 참고 사이트를 찾던 중, 다양한 기능이 구성되어 있는 웹사이트를 발견하였고, 그 기능들을 참고해 직접 만들어 보고 싶은 마음에 제작을 하게 되었습니다. 원래 사이트에는 없었던 로그인/회원가입 기능과 리뷰 게시판 기능을 추가하여 완성했습니다. 3개의 포트폴리오 중에서 기능 구현에 가장 초점을 많이 둔 포트폴리오입니다.

제작에 사용한 기술

HTML/CSS JavaScript Node.js MongoDB

제작 기간

14일

반응형 작업

상세 설명 (메인 페이지)

LG SIGNATURE의 글로벌 홈페이지를 구현해 보았습니다. 외국인 사용자를 대상으로 LG SIGNATURE의 전자제품을 소개 및 홍보하는 목적이기 때문에, 웹페이지 전체가 영어로 작성되어 있습니다. 메인 페이지에서는 각각의 제품들을 구경할 수 있도록 경로를 설정해두었으며, LG SIGNATURE의 대표 제품인 Rollable OLED TV를 주력으로 소개하고 있습니다.

상세 설명 (로그인/회원가입 페이지)

MongoDB의 데이터베이스 서비스를 이용해 로그인/회원가입 기능을 추가해 보았습니다. 로그인 페이지에서는 가입한 아이디 및 비밀번호를 정확히 입력해야 로그인이 가능하도록 하였고, 회원가입 페이지에서는 정규표현식을 사용해 회원정보 기입 시 잘못된 입력을 피할 수 있도록 만들었습니다.

상세 설명 (리뷰 게시판 페이지)

게시판의 형식으로 만든 리뷰 페이지는 고객들이 직접 글을 작성하고 수정/삭제까지 가능하도록 만들어 보았습니다. 로그인을 해야만 리뷰 게시판 페이지로 접근이 가능하도록 만들었으며, 관리자 아이디로 접속했을 경우, 모든 글을 수정하고 삭제할 수 있게 설정했습니다. 게시판 페이징 기능과 검색 기능도 함께 추가해 좀 더 사용하기에 편리할 수 있도록 만들어 보았습니다.

  • 메인 페이지
  • 로그인/회원가입 페이지
  • 리뷰 게시판 페이지

Portfolio 02

Nest Hotel Tokyo Hanzomon 포트폴리오 보러가기

제작 의도

앞선 포트폴리오에선 기능에 초점을 두었다면, 이번에는 웹사이트의 목적과 역할 그 자체에 좀 더 집중해 보고 싶었습니다. 그래서 호텔 예약을 위한 웹사이트를 참고해 로그인/회원가입 및 호텔 예약 서비스를 구현해 보았습니다. 실제로 예약 날짜와 인원 및 방 종류 등을 선택할 수 있고, 예약이 완료되면 예약 내역 정보까지 확인해 볼 수 있도록 만들었습니다.

제작에 사용한 기술

HTML/CSS JavaScript Node.js MongoDB

제작 기간

7일

반응형 작업

상세 설명 (메인 페이지)

일본 호텔인 Nest Hotel Tokyo Hanzomon의 호텔 소개 및 예약 사이트를 구현해 보았으며, 이 사이트 역시 호텔을 찾아온 외국인을 대상으로 했기 때문에, 모든 페이지가 영어로 작성되었습니다. 메인 페이지에서는 방의 종류, 조식 여부, 찾아오시는 방법 등 호텔과 관련된 정보들을 전반적으로 소개하는 섹션들로 구성했습니다.

상세 설명 (호텔 예약 페이지)

로그인 후 이용이 가능한 호텔 예약 페이지에는 체크인/체크아웃 날짜 선택, 투숙객 인원 수, 방의 갯수와 종류 선택 등의 기능을 추가해 보았습니다. 날짜 선택 기능을 구현할 때는 달력을 직접 화면에 출력해보고 싶어서 flatpickr이라는 라이브러리를 사용했습니다.

상세 설명 (예약 정보 확인 페이지)

메인 페이지에서 로그인 후에만 사용할 수 있는 My Booking 페이지로 접속하게 되면, 사용자가 예약한 호텔 예약 정보를 확인 할 수 있습니다. 사용자가 여러 날짜에 예약을 하더라도 각각의 정보를 열람할 수 있으며, 예약 상세 페이지에는 예약을 취소하는 기능도 추가되어 있습니다.

  • 메인 페이지
  • 호텔 예약 페이지
  • 예약 정보 확인 페이지

Portfolio 03

Cafe Paul Bassett 포트폴리오 보러가기

제작 의도

마지막 포트폴리오에서는 휠(wheel)이벤트로 한 섹션씩 이동하는 모션을 메인페이지에 적용해보고 싶었습니다. API를 사용하지 않고 jQuery를 이용해 구현을 시도해보았으며, 여기에도 로그인/회원가입 기능과 함께 QNA 페이지를 추가해 사용자가 질문을 하면 관리자가 답변을 할 수 있도록 만들어 보았습니다.

제작에 사용한 기술

HTML/CSS JavaScript jQuery Node.js MongoDB

제작 기간

10일

반응형 작업

상세 설명 (메인 페이지)

카페 Paul Bassett을 소개하는 페이지로, 한 섹션마다 각각의 주제에 맞는 내용들로 구성되어 있습니다. 처음 포트폴리오를 만들 때부터 해보고 싶었던 휠 이벤트로 한 섹션씩 이동하는 기능을 구현해 보았으며, 아코디언 메뉴, 탭 메뉴, 캐러셀, 슬라이드 등을 적절히 배치해 보았습니다.

상세 설명 (카페 메뉴 페이지)

Paul Bassett의 메뉴 소개 페이지입니다. 분류 별로 커피, 음료, 아이스크림 등으로 나누어 정리되어 있습니다. NEW 메뉴에서는 isotope API를 적용하여 새 소식 정보를 카테고리 별로 분류 및 정리하여 볼 수 있게 해 두었습니다.

상세 설명 (QNA 페이지)

로그인을 해야 이용할 수 있는 QNA 페이지는 사용자가 문의하고 싶은 사항을 직접 작성해서 올릴 수 있도록 만들어 보았습니다. 수정 및 삭제 또한 가능하며, 질문을 작성하면 추후에 관리자가 그에 대한 답변을 작성할 수 있게 했습니다. 관리자 답변창은 관리자만 사용할 수 있으며, 관리자가 답변을 하지 않은 상태라면 답변 대기 중이라는 문구가 답변 대신 등장합니다.

  • 메인 페이지
  • 카페 메뉴 페이지
  • QNA 페이지
